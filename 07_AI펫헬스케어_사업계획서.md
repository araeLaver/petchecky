## 2. 문제 정의

### 2.1 핵심 문제
```
"우리 강아지가 갑자기 토했는데... 병원 가야 하나?"
```

### 2.2 현재 사용자 경험
```
반려동물 이상 증상 발견
        ↓
인터넷 검색 → 온갖 무서운 병명 → 불안 증폭
        ↓
결정 못하고 시간 지체 OR 과잉 병원 방문
        ↓
병원 방문 후 기록 분산, 관리 안 됨
```

### 2.3 Pain Point 상세

| Pain Point | 심각도 | 빈도 | 대상 |
|------------|--------|------|------|
| 증상 판단 어려움 | ⭐⭐⭐⭐⭐ | 높음 | 전체 |
| 병원비 부담 | ⭐⭐⭐⭐⭐ | 중간 | 전체 |
| 야간/주말 응급 대응 | ⭐⭐⭐⭐ | 낮음 | 전체 |
| 진료 기록 분산 | ⭐⭐⭐⭐ | 높음 | 2년+ 보호자 |
| 좋은 병원 찾기 | ⭐⭐⭐⭐ | 중간 | 신규/이사 |

### 3.2 핵심 기능

#### MVP (Phase 1)
| 기능 | 설명 | 기술 |
|------|------|------|
| 펫 프로필 | 종, 나이, 체중, 질병 이력 | DB |
| AI 증상 체커 | 텍스트 기반 증상 분석 | LLM + 의료 DB |
| 위험도 판단 | 녹색/노랑/빨강 단계 구분 | Rule Engine + ML |
| 대응 가이드 | 상황별 행동 지침 | 콘텐츠 DB |
| 병원 추천 | 위치 기반 동물병원 검색 | 지도 API |

#### Growth (Phase 2)
| 기능 | 설명 |
|------|------|
| 사진 분석 | 피부, 눈, 대변 등 이미지 분석 |
| 진료 기록 | 병원 방문, 처방, 검사 결과 저장 |
| 예방접종 알림 | 일정 관리 및 리마인더 |
| 수의사 상담 | 텍스트/화상 상담 연결 |

#### Scale (Phase 3)
| 기능 | 설명 |
|------|------|
| 펫보험 연동 | 보험 비교, 가입, 청구 |
| 맞춤 커머스 | 건강 기반 사료/용품 추천 |
| 웨어러블 연동 | 활동량, 수면 데이터 연동 |
| 커뮤니티 | 같은 질병/품종 보호자 연결 |

## 4. 시장 분석

### 4.1 시장 규모

| 시장 | 규모 (2024) | 성장률 | 2027 전망 |
|------|-------------|--------|-----------|
| 국내 반려동물 전체 | 6조원 | 10%+ | 10조원 |
| 펫헬스케어 | 1.2조원 | 15% | 2조원 |
| 펫테크 (앱) | 3,000억원 | 20% | 6,000억원 |

### 4.2 TAM-SAM-SOM

```
TAM (Total Addressable Market)
└── 국내 반려동물 시장: 6조원

    SAM (Serviceable Addressable Market)
    └── 펫 헬스케어 + 테크: 1.5조원

        SOM (Serviceable Obtainable Market)
        └── 앱 기반 건강관리: 500억원
            └── 3년 목표 점유율 20%: 100억원
```

### 4.3 시장 트렌드

| 트렌드 | 설명 | 기회 |
|--------|------|------|
| 펫팸족 증가 | 반려동물 = 가족 | 프리미엄 케어 수요 |
| 1인 가구 증가 | 반려동물 = 동반자 | 불안 해소 니즈 |
| 고령화 | 반려동물도 고령화 | 만성질환 관리 |
| 디지털 전환 | MZ 보호자 증가 | 앱 기반 서비스 수용 |

## 5. 경쟁 분석

### 5.1 경쟁사 비교

| 항목 | 펫닥 | 핏펫 | 강아지궁합 | **펫닥터AI** |
|------|------|------|-----------|-------------|
| 주요 기능 | 수의사 상담 | 건강+쇼핑 | 커뮤니티 | AI 증상 분석 |
| 강점 | 전문가 연결 | 올인원 | 활발한 커뮤니티 | 즉각적 AI 답변 |
| 약점 | 비용 부담 | 복잡한 UX | 기능 제한 | 신규 서비스 |
| MAU | 30만 | 50만 | 20만 | 목표 50만 |
| 수익모델 | 상담료 | 구독+커머스 | 광고 | 구독+상담+보험 |

### 5.2 경쟁 우위

| 차별점 | 설명 |
|--------|------|
| **즉각성** | 24시간 즉시 AI 분석 (상담 대기 X) |
| **저비용** | 월 5,900원으로 무제한 사용 |
| **개인화** | 우리 아이 데이터 기반 맞춤 분석 |
| **통합성** | 증상→분석→병원→기록 원스톱 |

### 5.3 경쟁 장벽 (Moat)

```
1. 데이터 → 사용자 증상/진료 데이터 축적
              ↓
2. AI 정확도 → 데이터 기반 분석 정확도 향상
              ↓
3. 신뢰 → 정확한 분석으로 사용자 신뢰 구축
              ↓
4. 네트워크 → 병원 제휴, 커뮤니티 형성
              ↓
5. 전환 비용 → 축적된 기록, 습관화
```
