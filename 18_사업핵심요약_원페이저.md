## 🎯 무슨 서비스?

### 1번 서비스: AI 펫 헬스케어

```
"반려동물이 아파 보일 때, AI가 즉시 답해주는 건강 상담 앱"
```

| 항목 | 내용 |
|------|------|
| 한 줄 설명 | 강아지/고양이 증상을 말하면 AI가 원인 분석 + 병원 가야 할지 판단 |
| 타겟 | 반려동물 보호자 1,500만명 (국내) |
| 핵심 문제 | "우리 강아지가 아파 보이는데 병원 가야 하나?" → 검색해도 답 없음 |
| 핵심 해결 | AI가 증상 분석해서 즉시 가이드 제공 |

### 2번 서비스: AI 육아 코파일럿

```
"육아 질문에 AI가 24시간 즉시 답해주는 앱"
```

| 항목 | 내용 |
|------|------|
| 한 줄 설명 | 아기 증상/육아 질문하면 AI가 맞춤 답변 + 수유/수면 기록 |
| 타겟 | 0-36개월 자녀 부모 120만명/년 |
| 핵심 문제 | "새벽 3시 아기가 아파 보이는데 누구한테 물어보지?" |
| 핵심 해결 | AI가 내 아이 상황에 맞게 즉시 답변 |

## 🏆 왜 우리가 이길 수 있나?

### 경쟁사 vs 우리

| 비교 | 핏펫 | 펫닥 | 베이비빌리 | **우리** |
|------|------|------|------------|----------|
| AI 상담 | △ 부정확 | ✗ 없음 | ✗ 없음 | **◎ 핵심** |
| 즉시 답변 | ✗ | ✗ 30분+ 대기 | ✗ | **◎ 즉시** |
| 가격 | 무료~9,900원 | 15,000원/회 | 5,900원 | **5,900원** |
| 전문성 | △ 커머스 중심 | ◎ 수의사 | △ 맘카페 수준 | **◎ AI+전문가** |

### 우리만의 차별점

```
1. AI 품질: Claude 기반 고품질 의료 상담
2. 즉시성: 24시간 즉시 답변 (경쟁사는 대기 필요)
3. 가격: 무료~5,900원 (펫닥 대비 1/3 가격)
4. 맞춤화: 내 반려동물/아이 정보 기반 개인화
5. 기술 시너지: 펫+육아 동시 운영으로 비용 40% 절감
```

## 💵 돈 얼마 벌어?

### 현실적인 목표

| 시점 | MAU | 연매출 | 상태 |
|------|-----|--------|------|
| **1년차** | 3만명 | 1.6억원 | 적자 (투자금 소진) |
| **2년차** | 15만명 | 12억원 | 손익분기 근접 |
| **3년차** | 50만명 | 60억원 | 흑자 전환 |
| **3년차 (통합)** | 100만명 | 120억원 | 본격 성장 |

### 핵심 수익 공식

```
월 매출 = (MAU × 유료전환율 × 구독료) + (상담건수 × 수수료) + (보험가입 × 중개료)

예시 (Year 2):
= (150,000 × 7% × 5,900) + (5,000 × 4,500) + (500 × 70,000)
= 62,000,000 + 22,500,000 + 35,000,000
= 약 1.2억원/월 = 약 14억원/년
```
